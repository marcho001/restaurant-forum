## 安裝

- 開啟終端機，輸入
```
git clone https://github.com/marcho001/restaurant-forum.git
```
- 進入資料夾安裝套件
```
npm install
```
- 執行種子程式
```
npm run seed
```
- 執行程式
```
npm run dev
```
